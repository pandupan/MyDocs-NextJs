# 📄 MyDocs — Next.js

Pembelajaran fundamental **Next.js** secara bertahap mengikuti dokumentasi resmi. Menggunakan **Pages Router** dan **Next.js 13.4.7**.

## ✨ Mencakup

- Pages Router vs App Router
- Routing dan navigasi
- Data fetching
- API routes
- Middleware
- Deployment

## 🛠️ Tech Stack

- **Next.js 13.4.7** — React framework (Pages Router)
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
